== Description ==

Encurte as URLs do seu blog usando o http://ads.tt, o encurtador de URLs da boo-box! Ganhe dinheiro divulgando seus links pelas redes sociais!

== Installation ==

1. Suba a pasta  `booencurtador` na pasta `/wp-content/plugins/` do seu blog
2. Ative o plugin através do menu Plugins no Wordpress
3. Nas configurações, inclua o seu email cadastrado na boo-box
4. Pronto! O plugin vai criar uma URL encurtada sempre que você criar um novo post. Use essa URL nas redes sociais e ganhe dinheiro divulgando seus links!
